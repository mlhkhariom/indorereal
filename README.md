# Indore Realty — Premium Properties in Indore

**Indore Realty** is Indore's #1 trusted real estate platform with 500+ verified property listings. Built and maintained by **MLHK Infotech**.

## 🏠 Features

- **Property Listings** — Browse verified flats, plots, villas & houses across 15+ prime Indore locations
- **Advanced Search & Filters** — Search by location, type, budget, and status with real-time filtering
- **Dynamic Gallery** — Image gallery with category filtering, managed from admin panel
- **Blog & Articles** — SEO-optimized blog with real estate guides, market insights & tips
- **Lead Management** — Contact forms and enquiry tracking with status management
- **EMI Calculator** — Built-in home loan EMI calculator
- **Admin Panel** — Full CMS to manage properties, locations, gallery, blogs, homepage content & SEO settings

## 🔍 SEO & Structured Data

- JSON-LD schemas: `LocalBusiness`, `RealEstateAgent`, `Organization`, `FAQPage`, `BreadcrumbList`, `Product`
- Dynamic meta tags, OpenGraph & Twitter cards on every page
- Google's 2026 SEO rules compliant — INP optimized, lazy loading, semantic HTML

## 🛠️ Tech Stack

- **Frontend:** React 18, TypeScript, Vite, Tailwind CSS, Framer Motion
- **UI Components:** shadcn/ui (Radix primitives)
- **Backend:** Lovable Cloud (Supabase) — Database, Auth, Storage, Edge Functions
- **State Management:** TanStack React Query

## 📱 Mobile Responsive

Fully responsive across all devices — mobile-first design with bottom navigation bar, touch-optimized interactions.
